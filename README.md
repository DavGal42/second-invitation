# Հարսանեկան հրավիրատոմս — փոփոխությունների ուղեցույց

Այս կայքի բոլոր հիմնական տեքստերը, ամսաթիվը, վայրերը, քարտեզների հղումները և երաժշտության հասցեն հավաքված են մեկ ֆայլում՝ `app/invitation-content.ts`։ Դիզայնը փոխելու համար հիմնականում պետք չէ խմբագրել կայքի կառուցվածքը։

## Տեքստերի փոփոխություն

Բացեք `app/invitation-content.ts` և փոխեք չակերտների մեջ գրված արժեքները։

- `eyebrow` — վերևի «ՀՐԱՎԻՐԱՏՈՄՍ» գրությունը։
- `couple.first`, `couple.connector`, `couple.second` — զույգի անունները և «ԵՎ» բառը։
- `greeting` — հյուրերին ուղղված վերնագիրը։
- `introduction` — հրավերի կարճ տեքստը։
- `displayDate` — էկրանին երևացող ամսաթիվը, օրինակ՝ `02/08/26`։
- `eventDateTime` — հետհաշվարկի ճշգրիտ օրը և ժամը։ Պահեք ISO ձևաչափը, օրինակ՝ `2026-08-02T16:00:00+04:00`։ `+04:00`-ը Հայաստանի ժամային գոտին է։
- `countdownLabel` — հետհաշվարկի վերնագիրը։
- `directionsLabel` — քարտեզի կոճակի տեքստը։
- `closing` — էջի վերջի մեծ գրությունը։

Յուրաքանչյուր վայր `venues` ցուցակում ունի հետևյալ դաշտերը.

- `heading` — բաժնի վերնագիրը։
- `name` — եկեղեցու կամ սրահի անունը։
- `time` — միջոցառման ժամը։
- `mapEmbedUrl` — կայքի ներսում բացվող քարտեզը։
- `mapLink` — Google Maps-ի առանձին էջի հղումը։

## Լուսանկարների և պատկերների ավելացում

Կայքում հիմա կան երեք դիտավորյալ դատարկ պատկերային տարածքներ՝ զույգի գլխավոր լուսանկարը, եկեղեցու պատկերը և հանդիսության վայրի պատկերը։ Դրանք պահում են օրինակի ճիշտ բարձրությունն ու տարածությունը։

1. Տեղադրեք պատկերները `public/images` թղթապանակում, օրինակ՝
   - `public/images/couple.jpg`
   - `public/images/church.jpg`
   - `public/images/reception.jpg`
2. Բացեք `app/page.tsx`։
3. Գտեք `hero-image-slot` և երկու `venue-image-slot` դատարկ բլոկները։
4. Համապատասխան `<div ... />` բլոկը փոխարինեք `<img>` տարրով, օրինակ՝

```tsx
<img
  className="image-slot hero-image-slot"
  src="/images/couple.jpg"
  alt="Աշոտ և Մարիամ"
/>
```

Վայրերի պատկերների համար օգտագործեք նույն ձևը՝ թողնելով `venue-image-slot` դասը և փոխելով միայն `src` ու `alt` արժեքները։ `app/globals.css` ֆայլում `.image-slot` կանոնի մեջ կարող եք ավելացնել `object-fit: cover;`, եթե լուսանկարը պետք է ամբողջությամբ լցնի տարածքը։

## Քարտեզների փոփոխություն

Google Maps-ում գտեք անհրաժեշտ վայրը և պատճենեք դրա հասցեն կամ անունը։

- Ներկառուցված քարտեզի համար օգտագործեք `https://www.google.com/maps?q=ՎԱՅՐԻ_ԱՆՈՒՆԸ&output=embed` ձևաչափը և տեղադրեք այն `mapEmbedUrl` դաշտում։ Բացատներն ու հայերեն տառերը ցանկալի է URL-կոդավորել։
- Արտաքին հղման համար օգտագործեք `https://www.google.com/maps/search/?api=1&query=ՎԱՅՐԻ_ԱՆՈՒՆԸ` ձևաչափը և տեղադրեք այն `mapLink` դաշտում։

«Ինչպես հասնել» կոճակը սահուն բացում և փակում է քարտեզը հենց հրավիրատոմսի ներսում։ Քարտեզի տակ գտնվող հղումը այն բացում է Google Maps-ում։

## Երաժշտության փոփոխություն

Կայքում ներառված է մեղմ սկզբնական մեղեդի՝ `public/music/background.wav`։ Բրաուզերների կանոնների պատճառով երաժշտությունը սկսվում է հյուրի առաջին հպումից կամ վերևի աջ անկյունում գտնվող երաժշտության կոճակից։ Կոճակը նաև դադարեցնում և կրկին միացնում է մեղեդին։

Ձեր երաժշտությունը տեղադրելու համար՝

1. Պատճենեք MP3 կամ WAV ֆայլը `public/music` թղթապանակ։
2. Բացեք `app/invitation-content.ts`։
3. `music.file` դաշտը փոխեք նոր ֆայլի հասցեով, օրինակ՝ `/music/our-song.mp3`։
4. `music.title` դաշտում գրեք երգի անունը։

Խորհուրդ է տրվում օգտագործել սեղմված MP3, որպեսզի կայքը արագ բացվի հեռախոսով։ Համոզվեք, որ երաժշտությունն օգտագործելու իրավունք ունեք։

## Տեղային գործարկում

```bash
npm install
npm run dev
```

Այնուհետև բացեք տերմինալում նշված տեղային հասցեն։ Վերջնական տարբերակը ստուգելու համար օգտագործեք `npm run build`։
